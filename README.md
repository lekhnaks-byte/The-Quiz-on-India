# The Quiz On India 
An interactive, responsive, and educational web application designed to test your knowledge about India.-
# Feautures ✨
* User Interaction - Users can choose an answer from the MCQ
* Instant feedback - Marks the option "green" if correct and "red" if incorrect
* Score Calculation - Result updates instantly as you play
# Tech Stack ⚒️
* HTML 5 - Structured Quiz layout
* Tailwind CSS - Used for modern, utility-first styling and smooth transitions
* JavaScript (ES6+) - DOM Manipulation, Event Delegation
